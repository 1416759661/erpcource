第一、我们安装了一个git这样一个工具。这个工具就是版本控制器。
第二、我们在github.com这个网站上注册了一个账号。这个网站就是用来托管代码的。我们可以把自己的代码传到这个网站上。
这个网站给我们提供免费的空间。管理代码的时候，这些代码是对外公开的。如果你想不公开你的代码，主要收费。


操作步骤：

第一步、安装git工具。
第二步、打开git bash这个命令窗口，在里面配置git账号。

git config --global user.name "tangyan"       //此处的名字可以自己定义。
git config --global user.email "1416759661@qq.com"   //这个账号必须是你注册成功，可以登录的github这个网站的账号。


它的作用就是标记后面一系列的操作都是这个人操作的。


常用命令： 
git status  这个命令可以随时查看版本库的状态。
git add test01.txt  表示把工作区修改过的文件test01.txt提交到版本库的暂存区。 
git add .  一个点表示把当前工作区文件夹下面的全部未提交的文件提交到暂存区。
git commit -m "这是为的第一个文件"  这个命令表示把暂存区的代码提交到主分支。  特别注意要加参数m 。
clear 命令可以清楚屏幕上的文字

git reset --hard  HEAD^  可以退回一个版本。如果要退回多个版本，可以加多个^ 这个符号。
如果要用数字指定退回的版本数量 可以使用 HEAD~100
如果要跳转到指定的版本，可以使用git reset --hard  6b1de8ac1      "6b1de8ac1"表示部分版本号
git diff test01.txt 此命令可以看到某个特定文件版本的变化，到底改变哪些东西。
